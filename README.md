# 🏗️ Industrial Defect Inspector v2.0

## 🎯 AI-powered construction defect analysis

**Production-ready application for construction defect inspection using Google Gemini AI**

### ✅ Features:
- 🤖 **Gemini AI Analysis** - Advanced defect detection
- 📱 **Mobile Application** - React Native with Expo
- 🌐 **Web API** - FastAPI backend
- 🗄️ **Database** - Supabase integration
- 🐳 **Docker Ready** - Production deployment
- 📊 **Reports** - Professional documentation

### 🚀 Quick Start:

#### **Server Deployment:**
```bash
git clone https://github.com/lacrenol/industrial-inspector-v2.git
cd industrial-inspector-v2/production
docker-compose up -d
```

#### **Mobile Development:**
```bash
cd mobile
npm install
npx expo start
```

#### **API Testing:**
```bash
curl -f http://your-server:8000/api/health
```
